# 📊 Sales Data Analysis using R

## 📌 Project Overview

Sales Data Analysis using R is a beginner-friendly data analytics project developed using **R Programming**. The project demonstrates how business sales data can be analyzed using statistical functions and visualized through charts to support better decision-making.

The application reads monthly sales data from a CSV file, performs calculations such as total sales, average sales, highest sales, and lowest sales, and generates graphical representations including bar charts and pie charts. The project is designed to introduce the fundamentals of data analysis and visualization while solving a practical business problem.
